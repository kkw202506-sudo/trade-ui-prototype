# 交易复盘系统 UI 原型

## 📊 项目简介

为个人A股散户设计的交易复盘系统UI原型，通过结构化录入、数据统计、AI智能分析帮助用户识别交易弱点，建立成长型复盘习惯。

### 在线预览

[GitHub Pages 链接部署中...]

### 核心功能

- 📝 **交易录入**：40秒内完成交易记录
- 📈 **统计分析**：资金曲线、策略胜率、月度盈亏
- 🎯 **策略标签**：趋势跟随、突破买入、反弹做多等
- 💡 **智能归因**：平仓后自动触发归因分析
- 📱 **响应式设计**：支持移动端、平板、桌面

### 设计特点

- 🇨🇳 **A股本地化**：红涨绿跌配色（符合中国市场习惯）
- ♿ **可访问性**：WCAG AA标准，键盘导航、屏幕阅读器支持
- 🎨 **专业设计**：完整的组件库和设计系统
- ⚡ **高性能**：纯HTML/CSS/JS，无外部依赖

## 🚀 快速开始

### 本地预览

```bash
# 克隆仓库
git clone https://github.com/你的用户名/trade-ui-prototype.git
cd trade-ui-prototype

# 使用浏览器打开 ui-prototype.html
# 或者使用本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000/ui-prototype.html
```

### 在线预览

直接访问 GitHub Pages 链接即可（部署后会更新）

## 🎨 设计系统

### 配色方案

- **盈利色（红色系）**：#ef4444（符合A股红涨习惯）
- **亏损色（绿色系）**：#22c55e（符合A股绿跌习惯）
- **主色调（蓝色系）**：#3b82f6

### 组件库

- 按钮（主要、次要、幽灵，3个尺寸）
- 表单（输入框、文本域、选择器）
- 卡片（标准卡片、统计卡片）
- 标签（5种颜色变体）
- 表格、导航、空状态等

### 可访问性

- 色彩对比度 ≥ 4.5:1
- 键盘完整导航支持
- 触摸目标 ≥ 44×44px
- 支持屏幕阅读器

## 📱 页面列表

1. **首页/看板**：统计卡片、资金曲线、策略胜率、最近交易
2. **交易录入**：完整的表单系统，支持标签选择
3. **统计分析**：时间范围切换、图表矩阵、策略分析
4. **交易详情**：盈亏摘要、交易详情、归因分析

## 🛠️ 技术栈

- 纯 HTML5 + CSS3 + 原生 JavaScript
- 无外部依赖，可直接运行
- 响应式设计（移动端、平板、桌面）
- 符合 WCAG AA 可访问性标准
